# Simulado Interativo PC-PR 2026 - FGV

Arquivos gerados:

- `index.html`: versão interativa completa, funciona offline no navegador.
- `style.css`: estilo visual profissional.
- `script.js`: banco de questões e lógica de clique, correção, explicação e pontuação.
- `simulado_pcpr_fgv_interativo.pdf`: versão PDF com botões/links internos para navegar e acessar comentários.
- `gabarito_pcpr_fgv.pdf`: gabarito comentado separado.

## Como usar

1. Abra `index.html` no navegador.
2. Responda às questões clicando em A, B, C, D ou E.
3. Clique em “Ver explicação” somente após responder.
4. Veja sua pontuação e o mapa de erros.
5. Use o PDF para estudo em leitor de PDF. O melhor suporte de links e botões costuma ocorrer no Adobe Acrobat Reader.

## Observação técnica

A interatividade mais confiável está no HTML. A maioria dos leitores de PDF limita JavaScript e campos dinâmicos; por isso, o PDF foi construído com links internos e botões clicáveis para comentários e gabarito.

## Base de estilo

Questões inéditas, inspiradas no estilo FGV observado em provas policiais recentes: PC-PI Oficial Investigador, PC-MG Investigador, PC-RJ Investigador, PC-AM Investigador e PC-RN Agente/Escrivão. Não há cópia literal de questão oficial.
