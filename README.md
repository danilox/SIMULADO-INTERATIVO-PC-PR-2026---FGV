# Simulado Interativo PC-PR 2026 — FGV

Simulado autoral e interativo para estudo da **Polícia Civil do Paraná — PC-PR 2026**, cargo de **Agente de Polícia Judiciária**, com foco no padrão da banca **FGV**.

## Estrutura

- **50 questões inéditas**
- **25 questões de Língua Portuguesa**
- **25 questões de Informática, Segurança da Informação, Internet, Sistemas, Malware e Crimes Digitais**
- Correção automática no navegador
- Comentário aparece logo abaixo da questão
- Botão para ver/ocultar comentário
- Mapa de desempenho por disciplina

## Provas usadas como base de estilo

O simulado foi inspirado no padrão das seguintes provas FGV, sem copiar questões oficiais:

- PC-PI — Oficial Investigador
- PC-MG — Investigador de Polícia I
- PC-RJ — Investigador Policial
- PC-AM — Investigador de Polícia
- PC-RN — Agente e Escrivão de Polícia Civil
- Provas FGV de Perito/Informática Forense apenas como apoio para temas de tecnologia e segurança

## Como publicar no GitHub Pages

1. Suba todos os arquivos na raiz do repositório.
2. Confirme que o arquivo `index.html` está no mesmo nível do `README.md`.
3. Vá em **Settings → Pages**.
4. Selecione **Deploy from a branch**.
5. Escolha **Branch: main** e **Folder: /root**.
6. Clique em **Save**.

O site ficará disponível em:

```text
https://SEU-USUARIO.github.io/NOME-DO-REPOSITORIO/
```

## Aviso

Material autoral, independente, sem vínculo oficial com a FGV, PC-PR ou qualquer órgão público. Use como ferramenta de estudo e revisão.
